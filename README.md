HM Crisis Calculator

App web/PWA para iPhone y Android.

**Diseñada por Dr. Gonzalo Irizaga**

## Cómo publicarla en GitHub Pages

### Desde computadora
1. Entrar a https://github.com y crear una cuenta o iniciar sesión.
2. Crear un repositorio nuevo, por ejemplo: `hm-crisis-calculator`.
3. Subir todos los archivos de esta carpeta, no el ZIP completo:
   - `index.html`
   - `manifest.webmanifest`
   - `service-worker.js`
   - carpeta `icons/`
4. Ir a **Settings → Pages**.
5. En **Build and deployment**, elegir:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
6. Guardar.
7. GitHub entregará un enlace tipo:
   `https://TU-USUARIO.github.io/hm-crisis-calculator/`

## Cómo instalarla en iPhone

1. Abrir el enlace publicado en **Safari**.
2. Tocar el botón **Compartir**.
3. Elegir **Agregar a pantalla de inicio**.
4. Confirmar el nombre: **HM Crisis**.

## Cómo compartirla

Compartir el enlace publicado de GitHub Pages por WhatsApp, email o grupos clínicos. Cada usuario podrá abrirlo en Safari y agregarlo a su pantalla de inicio.
